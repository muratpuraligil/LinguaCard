Geliştirme sunucusunu başlatmak için aşağıdaki adımları uygula:

// turbo-all
1. Terminalde aşağıdaki komutu çalıştır:
   `npm run dev`

2. İşlem başladıktan sonra kullanıcıya uygulamanın şu linkte yayında olduğunu söyle:
   **http://localhost:5173**
   Ayrıca bu linke CMD + Tık (Mac) yaparak tarayıcıda hemen açılabileceğini de ekle.
